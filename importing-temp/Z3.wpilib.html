Installing WPILib VSCode

In order to begin programming your robot, you will need to install the WPI Library, which is a special version of Visual Studio Code with FRC-specific additions like the built-in ability to deploy code to the robot.

The official instructions for installing WPILib can be found [here](https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-2/wpilib-setup.html). This section is intended as additional information.

<div>

<label>

Download the WPILib installer from the link above. If you are using a non-Windows system, check the “Downloads for other platforms” link.

<label>

When the download is complete, launch FIle Explorer and find the downloaded file. It should be at the top of the “Downloads” folder by default. To properly run the installer, right click the file and select the “Mount” option with the icon of a disc. After the file is mounted, click WPILibInstaller.exe

</label>

<input type="checkbox"/>
