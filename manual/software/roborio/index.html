Imaging RoboRIO 2.0

The official instructions for imaging the roboRIO 2.0 can be found here. This section is intended as additional information.


The National Instruments roboRIO 2.0 included with the rookie Kit of Parts has to be “imaged” with a known set of files before it can be used to run your robot. You must have completed the TODO: FRC Game Tools installation and have your roboRIO connected to power from the Power Distribution Hub. For veteran teams with the original roboRIO, see the WPILib documentation for instructions on imaging the original roboRIO.


To image the roboRIO 2.0, the included 4 GB microSD card has to be inserted into a SD card reader in a computer. The Lenovo laptop included in the rookie Kit of Parts includes a built-in SD card slot, but an adapter is required to fit a microSD card.
